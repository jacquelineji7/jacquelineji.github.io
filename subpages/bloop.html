How it works

  • n this game, the player is the main character Bloop which is controlled using the mouse cursor. The goal of this game is to enter the Safezone. In order to go to the next level, the player would have to avoid all the viruses coming out from the Safezone and enter it. The higher the level, the more viruses being generated. If the player fails to avoid the viruses, Bloop dies and the game restarts.
  • There is also an additional object Pill which appears randomly throughout the game. Once the player "eats" the Pill, all the viruses slow down.
